# PlayStation Physical Disc Exit

## PlayStation Physical Disc Exit 事件概览

一句话定义：

2026-07-01，Sony Interactive Entertainment 宣布 2028-01 起 PlayStation 新发行游戏停止生产实体光盘，并在同日宣布 PS3 与 PS Vita 内置 PlayStation Store 分阶段关停，这把游戏购买从“可转让实体介质”进一步推向[[Platform-Mediated Ownership|平台中介所有权]]。

## 3. 触发因素

### 导火索

- 2026-07-01，PlayStation.Blog 发布公告：2028-01 起，所有在 PlayStation 主机上新发行的游戏不再生产实体光盘，后续新游戏以 PlayStation Store 和零售商数字格式销售。
- 同日，Sony 宣布 PS3 与 PS Vita 的内置 PlayStation Store 分阶段关闭：部分市场 2026-08 起关闭，其他国家在 2027-07 关闭；已购买内容据公告仍可在一段时间内继续下载。

### 根本原因

- [[Digital Distribution Lock-In]]：平台把发行、支付、许可、下载、更新和售后集中到账号系统中，提高控制力，也提高用户对平台持续运营的依赖。
- 数字发行降低实体制造、库存、物流和二手流通带来的成本与摩擦。
- 大型游戏容量、补丁、联网功能和 DRM 让“完整游戏在实体介质中独立存在”的模式越来越弱。
- 老旧商店的支付、安全、维护和合规成本上升，平台倾向把资源从旧硬件迁移到新平台。

## 4. 传导机制

```text
实体光盘停止用于新发行游戏
→ 玩家购买对象从可转让介质转为平台账号下的数字许可
→ 转卖、借用、收藏和离线保存能力下降
→ 平台获得更强定价、分发、数据和生命周期控制
→ 用户信任、保存权和退出选择受到冲击
→ 社区反弹、竞争平台迁移、盗版诱因或监管关注上升
→ 平台利润率改善与品牌信任折损同时出现
```

实际传导链：

- 该事件不是传统金融危机，但它是数字平台经济中的所有权重构事件：现金流、用户留存、渠道权力和品牌信任会通过平台规则变化重新分配。
- PS3/PS Vita 商店关停与 2028-01 新游戏数字化绑定在同一天出现，使“数字版未来可访问性”成为核心风险感知，而不是单纯的介质升级。

## 5. 主要影响

| 影响对象 | 结果 |
|---|---|
| Sony / PlayStation | 可能提升数字分发占比、减少实体渠道摩擦，并强化账号与商店入口；同时承担更高的用户信任和长期访问责任。 |
| 游戏发行商 | 减少光盘制造与零售库存成本，削弱二手市场；但若用户反弹扩大，可能影响预购、首发口碑和平台选择。 |
| 玩家 / 居民部门 | 转卖、借用、收藏、离线保存和“不依赖账号”的权利弱化。 |
| 实体零售 / 二手市场 | 新游戏光盘供给在 2028-01 后消失，实体零售可能转向数字码、周边或存量二手交易。 |
| 竞争平台 | Steam、Xbox、Nintendo、PC 生态和盗版渠道可能被重新比较；用户是否迁移取决于价格、便利性、保存权和社交网络。 |
| 游戏保存 | 旧商店关闭和新游戏数字化叠加，强化长期保存风险。 |
| 金融市场 | 暂未记录股价或财报反应；需要后续补充 SIE 数字销售占比、毛利率、用户流失和平台活跃数据。 |

## 8. 投资启发

- 数字化不只是渠道变化，也是在重写用户、平台、发行商和零售商之间的权利分配。
- 平台如果削弱用户可感知的所有权，需要用价格、便利性、跨设备体验或长期访问承诺补偿，否则“利润率提升”可能被品牌信任折损抵消。
- 旧商店关停是数字资产的压力测试：当平台生命周期进入尾部，用户才会真正意识到自己购买的是可撤销或受条件约束的访问权。
- 对平台公司而言，[[Digital Distribution Lock-In]]既可能成为利润护城河，也可能成为用户迁移和监管叙事的触发器。

## 9. 未解决问题

- PlayStation 停止实体光盘后，是否会提供游戏转赠、家庭共享、二手数字许可、离线备份或 disc-to-digital 转换机制？
- 用户反弹会停留在评论区，还是会影响硬件销量、软件购买、订阅续费和平台活跃？
- Steam 与 PlayStation 的差异是否在于“同样数字化，但用户预期、价格结构和开放硬件生态不同”？
- Web3 式“用户真正拥有数字资产”的叙事，是否会因中心化平台关停商店而获得真实需求，还是仍受体验、成本和版权约束限制？
- 需要补充 Sony / SIE 数字销售占比、实体盘收入、二手市场规模、用户留存和股价反应数据。
