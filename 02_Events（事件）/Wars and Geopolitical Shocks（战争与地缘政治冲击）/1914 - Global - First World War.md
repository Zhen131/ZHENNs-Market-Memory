# 1914 - Global - First World War

## 事件概览

一句话定义：

第一次世界大战是一场由 1914 年巴尔干危机触发、经联盟承诺与军事动员升级、最终扩展为全球工业化总体战的系统性冲击；它不仅摧毁了人口与帝国，也通过税收、债务、中央银行信用、通胀和国际借款重塑了国家资产负债表与世界货币秩序。

## 2. 故事主线：一场局部危机如何吞没旧世界

### 第一幕：萨拉热窝的枪声不是全部原因

1914 年 6 月 28 日，奥匈帝国皇储弗朗茨·斐迪南在萨拉热窝遇刺。刺杀提供了导火索，却没有自动决定世界大战必然发生。此后的七月危机中，奥匈帝国在德国支持下向塞尔维亚施压并宣战；俄国为支持塞尔维亚而动员；德国先后对俄国和法国宣战并进攻比利时；英国随后参战。

真正让局部冲突失控的，是一系列具体决策与结构条件叠加：民族主义、军备竞赛、帝国竞争、对盟友信誉的担忧、铁路动员计划的刚性，以及各国领导人对短期战争的误判。

### 第二幕：速胜计划失败，战争变成工业消耗

德国未能迅速击败法国，西线逐渐形成堑壕对峙。机枪、重炮、铁丝网、铁路和工业化后勤，使防御与持续补充变得异常重要。战争不再只是前线将领之间的战术较量，而是整套国家机器的竞赛：谁能持续征兵、生产炮弹、运输粮食、取得能源、维持进口、控制海运，并让居民继续接受牺牲。

在东线、巴尔干、中东、非洲与海上，战争同样展开。英国及其帝国的参战把殖民地兵员、全球航线和海外资源卷入战场；奥斯曼帝国加入同盟国后，苏伊士、达达尼尔与中东成为战略焦点。因此，一战不能被缩成“法国与德国在西线打堑壕战”。

### 第三幕：战争开始吃掉财政与货币制度

各国原本普遍低估战争会持续多久、需要多少钱。当军费远超正常税收时，政府必须在增税、发债、对外借款和货币扩张之间组合融资。[[Government Bonds]] 把居民储蓄和银行资产转化为国家购买武器、粮食与军需的能力；中央银行则越来越需要支持国债市场与财政融资。

与此同时，[[Gold Standard]] 的可兑换约束与战时流动性需求发生冲突。多个交战国暂停或弱化黄金兑换，让货币与信贷更容易扩张。国家由此获得继续作战的空间，但成本并没有消失，而是以公共债务、通胀、汇率压力和战后调整的形式延后。

### 第四幕：1917 年的双重转折

俄国在战争损失、粮食困难、财政货币压力和政治危机中爆发革命。新的布尔什维克政府在 1917 年底与同盟国停战，并于 1918 年 3 月 3 日签署《布列斯特—立托夫斯克条约》，正式退出战争。德国得以把部分兵力转向西线，但 1917 年初恢复无限制潜艇战的高风险决定又推动美国参战。

美国此前保持中立，却已经通过贸易与金融同协约国深度连接。德国恢复无限制潜艇战，加上齐默尔曼电报引发的安全担忧，促使美国于 1917 年 4 月对德宣战。美国的工业产能、信贷与随后抵达欧洲的军队，使协约国获得新的持续作战能力。

### 第五幕：1918 年停火，但账没有结清

到 1918 年秋，同盟国军队、财政、粮食供应和国内政治都接近极限。德国盟友相继退出战争，德国国内也出现动荡。1918 年 11 月 11 日停战协定生效，结束了西线主要战斗；它不是完整和平的同义词，欧洲其他地区的冲突与革命仍在继续。

1919 年的巴黎和会与《凡尔赛条约》开始重新划分领土、安全安排与赔款责任。战争期间形成的政府债务、盟国间债务、德国赔款、通胀与货币恢复问题，则把一战的经济后果延伸到整个战间期。

## 3. 发生前的环境

| 维度 | 状态 |
|---|---|
| 国际政治 | 欧洲列强在联盟、民族主义、帝国竞争与安全困境中相互戒备。 |
| 军事能力 | 征兵制、铁路动员、重工业与现代武器使大规模快速动员成为可能，但也让动员一旦启动就更难逆转。 |
| 金融体系 | 战前国际金融以 [[Gold Standard]]、伦敦金融市场和跨境贸易信用为重要支柱。 |
| 财政预期 | 各国普遍没有为四年总体战预留足够税收与财政空间。 |
| 供应结构 | 欧洲工业国仍依赖国际航运、粮食、原材料与海外市场，海上封锁因此能转化为国内经济压力。 |
| 社会结构 | 国家可以动员数以千万计的士兵与大量平民劳动力，但长期伤亡、短缺与通胀会削弱社会承受力。 |

## 4. 触发因素与根本原因

### 导火索

- 1914-06-28，奥匈帝国皇储弗朗茨·斐迪南在萨拉热窝遇刺。
- 奥匈帝国在德国支持下向塞尔维亚发出最后通牒，并于 1914-07-28 宣战。
- 俄国动员、德国的战争计划、对比利时的入侵以及英国的参战，使局部战争迅速升级。

### 根本原因

- 联盟体系把局部安全承诺连接成列强之间的升级通道。
- 民族主义、帝国竞争与军备竞赛提高了接受战争的政治意愿。
- 军事动员计划强调速度与先发优势，压缩了外交回旋时间。
- 多国领导人错误估计战争会短促、可控，并低估工业化战争的消耗能力。

结论：一战既不是“只因刺杀而爆发”，也不是没有责任主体的机械事故；它是在高风险结构中，由一连串人类决策完成升级的战争。

## 5. 传导机制

### 政治—军事链条

```text
萨拉热窝刺杀
→ 奥匈帝国向塞尔维亚施压并宣战
→ 俄国动员支持塞尔维亚
→ 德国按战争计划对俄、法开战并入侵比利时
→ 英国及其帝国参战
→ 欧洲联盟冲突扩展为全球战争
```

### 财政—货币链条

```text
大规模征兵与工业化军需
→ 战争支出远超正常税收
→ 政府发行国债、短期票据并寻求海外借款
→ 银行体系和中央银行支持政府融资
→ 货币与信贷扩张，黄金可兑换受到限制
→ 通胀、汇率压力与公共债务上升
→ 战争成本由当期纳税人扩散到储户、债券持有人与未来纳税人
```

### 供给—社会链条

```text
劳动力转入军队与军工生产
→ 海运封锁、贸易中断与农业生产受扰
→ 粮食、能源和原材料短缺
→ 配给、价格管制与黑市扩大
→ 实际工资和现金购买力受压
→ 罢工、抗议、革命与政权稳定风险上升
```

这三条链不是彼此独立的。战场失败会削弱国债信用与国内信心；通胀和饥饿会破坏社会稳定；外部融资能力又会反过来决定一国还能维持多久的进口与军事行动。这正是 [[Wartime Fiscal-Monetary Expansion]] 的总体战版本。

## 6. 主要影响

| 影响对象 | 结果 |
|---|---|
| [[Government Bonds]] | 国债成为大规模动员居民储蓄和银行资产的工具；名义偿付承诺仍会受到通胀、汇率、违约与战后重组影响。 |
| 货币与黄金 | 多个交战国暂停或弱化黄金兑换；黄金、外汇与国际信用成为支持进口和汇率的战略资源。 |
| 银行系统 | 银行既销售和持有国债，也通过抵押融资与中央银行安排支持财政，国家信用与银行资产负债表更加绑定。 |
| 股票市场 | 1914 年战争冲击首先造成金融恐慌、市场停市与流动性压力；此后不同行业表现受军需、管制、税收和供应约束影响，不能用“战争必然利好股票”概括。 |
| 商品与物流 | 军需推高部分原材料需求，海运、封锁与供应中断造成短缺；价格上升不等于所有生产者都能自由获利。 |
| 居民现金与储蓄 | 通胀侵蚀现金和固定收益购买力，配给与短缺又限制名义收入转化为实际消费。 |
| 政府财政 | 公共债务与未来偿付责任显著扩大，停战后仍需处理退伍、重建、利息、债务和赔款。 |
| 国际金融中心 | 欧洲金融能力被战争削弱，美国因出口、黄金流入、对外信贷和美元使用扩大而提高国际金融地位。 |
| 政治版图 | 俄国、奥匈、奥斯曼与德意志帝国崩解或发生根本变化，欧洲出现新国家、边界争议与持续冲突。 |

## 7. 政策应对

| 机构 | 措施 | 效果与代价 |
|---|---|---|
| 政府 / 财政部门 | 增税、发行战争债券与短期票据、对外借款、扩大军需采购。 | 把私人资源转向战争，但形成长期债务、税负和代际分配问题。 |
| 中央银行 | 提供流动性、支持国债销售与抵押融资、稳定银行体系和汇率。 | 降低短期融资约束，同时削弱货币政策独立性并可能放大通胀。 |
| 外汇与货币当局 | 暂停或限制黄金兑换、管理跨境支付与资本流动。 | 保留黄金和外汇用于关键进口，但改变了战前货币承诺。 |
| 行政体系 | 实施配给、价格管制、贸易和航运控制、生产与劳动力动员。 | 维持军队和城市供应，同时产生短缺、寻租、黑市与分配冲突。 |
| 美国财政部与美联储 | 通过 Liberty Bonds、税收和银行体系为参战融资。 | 把新成立不久的美联储嵌入国债融资，并扩大美元与美国金融体系的国际角色。 |

## 8. 战后账本：为什么停战不等于风险消失

战后欧洲面对的不是一张账，而是相互嵌套的资产负债表：

```text
德国及其盟国的赔款义务
→ 英法等国希望用赔款支持重建并偿还战争债务
→ 美国坚持盟国偿还对美借款
→ 欧洲支付能力依赖出口、财政稳定与新的国际贷款
→ 任一环节受阻都可能转化为汇率、通胀、违约或政治冲突
```

这套结构后来与德国恶性通胀、鲁尔危机、道威斯计划、美国资本回流和大萧条发生联系。但应避免单因果叙事：一战债务与赔款是战间期脆弱性的核心部分，却不是二战爆发的唯一充分原因。

## 10. 投资与风险启发

- 战争首先是资源动员与资产负债表冲击，战场胜负不能脱离税收能力、债券市场、中央银行、外汇、进口与工业产能理解。
- 主权债券的“安全”必须区分名义偿付与实际购买力；国家可以还本付息，持有人仍可能因通胀或汇率变化遭受真实损失。
- 硬货币或固定兑换制度在和平时期提供纪律，但生存性冲击到来时，政府可能暂停规则；制度承诺本身也有状态依赖。
- “战争利好商品”或“战争利好军工股”都过于粗糙。需求上升可能同时伴随配给、税收、征用、价格管制、运输中断与市场关闭。
- 停火只是现金流停止恶化的第一步。债务、赔款、重建与货币修复决定战后是否形成稳定和平。

## 11. 未解决问题

- 需要按英国、法国、德国、俄国与美国分别补充战争支出中税收、国内债务、海外借款和货币融资的比例。
- 需要用统一数据比较各国通胀、汇率、国债实际回报与居民实际工资变化。
- 海上封锁、粮食配给与社会动荡之间的因果强度，需要更多国家级研究支持。
- 一战债务、赔款、恢复金本位与大萧条如何共同影响二战前政治激进化，需要在后续事件和对比笔记中拆分，避免“凡尔赛条约直接导致二战”的单线解释。
- 第二次世界大战入库后，需建立“一战 vs 二战”的正式 comparison 节点。
