# Europe Overview

## 定位

Europe 节点当前承接第一次世界大战及其财政、货币与战后秩序影响，不试图成为欧洲通史。这里把欧洲理解为一个由主权国家、帝国、金融中心、联盟体系与跨境债权债务共同构成的区域系统。

---

## 一战前的区域结构

| 维度 | 状态 |
|---|---|
| 政治结构 | 德意志、奥匈、俄国、奥斯曼等帝国与法国、英国、塞尔维亚等国家处于联盟、安全困境和民族主义压力中。 |
| 军事结构 | 征兵、铁路动员、军备竞赛和刚性作战计划提高了危机升级速度。 |
| 金融结构 | 伦敦是全球金融中心，[[Gold Standard]]、跨境银行与贸易信用支持资本和商品流动。 |
| 工业结构 | 德国、英国、法国等拥有大规模工业能力，但战争持续性仍依赖原料、粮食、航运与海外融资。 |
| 帝国结构 | 欧洲列强通过殖民地获取兵员、税收、资源和战略纵深，也把欧洲战争扩展到全球。 |

---

## 第一次世界大战如何改变欧洲

[[1914 - Global - First World War]] 同时改变了欧洲的四张地图：

- 政治地图：多个帝国崩解，新国家与新边界出现，但民族与领土争议没有随停战消失。
- 财政地图：交战国积累巨额公共债务，胜利国也并非财务上的无损赢家。
- 货币地图：战前 [[Gold Standard]] 秩序被打断，通胀、汇率和战后恢复平价成为长期政治问题。
- 债权地图：美国对欧洲盟国的贷款、盟国间债务与德国赔款形成相互嵌套的支付链。

```text
总体战摧毁与军费
→ 欧洲政府债务与进口需求上升
→ 对美国信贷和物资依赖加深
→ 战后赔款与盟国债务相互牵连
→ 欧洲金融稳定依赖跨国协调和持续资本流动
```
