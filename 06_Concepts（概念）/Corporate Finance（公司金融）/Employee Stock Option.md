# Employee Stock Option

## Employee Stock Option 概念概览

一句话定义：

Employee Stock Option 是公司授予员工在未来某段时间内、按约定价格购买公司股票或股份权益的条件性权利；它把员工报酬的一部分从现金工资转成对公司未来价值的索取权。

---

## 1. 核心字段

| 字段 | 内容 |
|---|---|
| 概念类别 | corporate-finance / equity-compensation |
| 主要作用 | 解释公司如何用未来股权收益吸引、保留和激励员工，也解释为什么上市前后容易出现激励兑现争议。 |
| 适用对象 | 创业公司、未上市平台公司、上市公司、核心员工、高管、早期投资者。 |
| 关键变量 | exercise price、vesting schedule、exercise window、公司估值、上市流动性、离职条款。 |
| 常见误用 | 把期权当成已经到手的现金；忽略归属条件、行权成本、离职窗口和未上市股份流动性。 |

---

## 2. 概念解释

- 员工期权不是直接发钱，也不是立即给股票，而是一份条件性购买权。
- 员工真正能否获益，取决于四件事：期权是否已经 [[Vesting and Exercise Conditions|归属]]、是否能在有效期内行权、行权价是否低于未来股票价值、以及股票是否有可出售的流动性。
- 对公司而言，[[Employee Stock Option]] 是一种把现金薪酬压力延后、把员工收益绑定到公司估值上的工具。
- 对员工而言，它同时是激励和风险：如果公司成功上市，期权可能变成重要财富；如果公司估值下跌、上市失败、离职窗口过短或合同解释不利，期权可能无法兑现。
- 在未上市公司中，期权价值更依赖 [[Initial Public Offering|IPO]]、并购或二级市场交易等流动性事件。没有流动性时，账面价值不等于可用现金。

---

## 3. 机制链条

```text
公司现金薪酬有限或希望绑定核心人才
→ 授予员工期权
→ 员工接受较长服务期、绩效条件或离职限制
→ 公司估值上升或接近 IPO
→ 期权从激励工具变成可计量财富
→ 合同条款、主体关系和离职安排成为利益冲突焦点
```

实际机制链：

- 期权的经济价值来自“未来股票价值”和“行权价”的差额。
- 期权的法律可得性来自合同条款，而不是员工主观贡献。
- 上市前夕，期权争议容易从劳动报酬问题扩大成 [[Pre-IPO Governance Risk|上市前治理风险]]，因为投资者和监管机构会关心潜在负债、员工诉讼和信息披露完整性。

## 4. 适用边界

| 条件 | 是否适用 / 限制 |
|---|---|
| 正常周期 | 适用于公司用长期激励替代部分现金薪酬的场景。 |
| 危机阶段 | 公司估值下跌或融资受阻时，期权激励可能失效，员工更关注现金补偿和离职权利。 |
| 高通胀环境 | 通胀不是核心变量，除非影响折现率、估值倍数和员工现金需求。 |
| 低通胀 / 通缩环境 | 若资本市场估值收缩，期权可能长期处于价外状态。 |
| 高杠杆环境 | 公司财务压力越大，越可能通过股权激励节省现金，但也更容易产生兑现争议。 |
| 政策干预强的环境 | 境内外主体、外资准入、上市地规则和劳动法差异会影响期权合同执行。 |

## 6. 与相近概念的区别

| 相近概念 | 相同点 | 关键区别 |
|---|---|---|
| [[Equity Incentive]] | 都属于股权相关激励。 | [[Equity Incentive]] 是总称，期权只是其中一种工具。 |
| restricted shares / RSU | 都把员工收益与公司价值绑定。 | 限制性股票更接近直接授予股权或股份单位，期权是未来按约定价格购买的权利。 |
| [[Vesting and Exercise Conditions]] | 都决定员工是否能拿到经济利益。 | 期权是权利本体；归属和行权条件是权利生效和使用的门槛。 |

## 8. 投资启发

- 看到高估值未上市公司时，要区分“公司估值上升”与“员工或投资者能否真正变现”。
- 期权纠纷不是小额劳动争议时，可能暴露公司治理、合同一致性、主体关系和潜在负债披露问题。
- 对拟上市公司，员工激励池、期权归属争议和离职安排会影响市场对治理质量的判断。
