# Platform-Mediated Ownership

## Platform-Mediated Ownership 概念概览

一句话定义：

Platform-Mediated Ownership 指用户名义上“买了”数字内容，但实际访问、转让、保存和持续使用都通过平台账号、商店、服务器和许可条款中介完成的所有权形态。

---

## 1. 核心字段

| 字段   | 内容                                                     |
| ---- | ------------------------------------------------------ |
| 概念类别 | platform-economics / digital-license / consumer-rights |
| 主要作用 | 解释数字内容购买中，为什么“支付了价格”不等于“拥有可自由处置的资产”。                   |
| 适用对象 | 平台公司、消费者、内容发行商、二手市场、监管机构                               |
| 关键变量 | 账号控制、许可条款、下载服务器、离线可用性、转售权、关停政策                         |
| 常见误用 | 把数字购买直接等同于实体购买；只看便利性，不看退出权和保存权。                        |

---

## 2. 概念解释

- 传统实体介质中，用户通常拥有一个可保存、转卖、借用、离线使用的物理对象。
- 数字许可中，用户通常获得的是在特定平台、特定账号、特定规则下访问内容的权利。
- 平台中介所有权的关键不在于“有没有付钱”，而在于用户能否在平台改变规则、关闭商店、封禁账号或停止服务器后继续控制该内容。
- 用户提到的 Web3 直觉也在这里：Web3 叙事强调用户拥有数据或数字资产，但中心化游戏平台的现实是平台仍掌握许可、分发和验证。

---

## 3. 机制链条

```text
内容从实体介质转向数字许可
→ 用户购买记录绑定到平台账号
→ 内容访问依赖商店、服务器、DRM 和许可条款
→ 转让、继承、备份、离线保存受到限制
→ 平台控制力和收入可预测性提高
→ 用户信任成为长期资产负债表之外的隐性负债
```

实际机制链：

- 在 [[2026 - Global - PlayStation Physical Disc Exit]] 中，实体光盘退出让新游戏从一开始就进入平台许可框架。
- 在 [[2025 - Global - Nintendo Switch 2 Game-Key Cards]] 中，物理卡仍存在，但内容本体需要下载，说明“看起来像拥有实体物”不等于“拥有完整内容”。

---

## 4. 适用边界

| 条件 | 是否适用 / 限制 |
|---|---|
| 正常周期 | 适用。用户通常只感受到便利性，不一定感知权利边界。 |
| 危机阶段 | 适用。商店关停、账号封禁、服务器停服时，权利边界会显性化。 |
| 高通胀环境 | 间接适用。数字平台可通过价格、订阅和版本策略转嫁成本。 |
| 低通胀 / 通缩环境 | 间接适用。折扣和低边际成本可能掩盖所有权弱化。 |
| 固定汇率制度 | 不相关。 |
| 浮动汇率制度 | 不相关。 |
| 高杠杆环境 | 对平台企业估值有间接影响：市场可能高估锁定带来的现金流稳定性。 |
| 政策干预强的环境 | 高度适用。消费者权益、数字转售、平台关停补偿和数据可携带性可能改变权利边界。 |

## 5. 历史验证事件

| 时间 | 事件 | 概念如何体现 | 验证了什么机制 |
|---|---|---|---|
| 2025 | [[2025 - Global - Nintendo Switch 2 Game-Key Cards]] | 物理卡作为下载钥匙，内容本体不在卡上。 | 物理凭证可以被平台许可化。 |
| 2026 | [[2026 - Global - PlayStation Physical Disc Exit]] | 新游戏停止实体光盘生产，购买转向数字格式。 | 用户所有权从介质控制转向账号控制。 |

## 6. 与相近概念的区别

| 相近概念 | 相同点 | 关键区别 |
|---|---|---|
| [[Digital Distribution Lock-In]] | 都讨论平台控制数字内容分发。 | 本概念强调“所有权形态”；模式节点强调“平台如何形成锁定和收益”。 |
| digital subscription | 都依赖平台访问。 | 订阅通常明确是租用；数字购买容易被用户误以为等同于拥有。 |
| DRM | 都限制复制和访问。 | DRM 是工具；Platform-Mediated Ownership 是由工具、账号、商店和规则共同形成的制度状态。 |

## 8. 投资启发

- 数字许可型平台的利润率提升，常常来自把用户权利边界重新写入平台规则。
- 平台估值分析需要把“用户是否相信自己买到的内容长期可用”当作关键变量。
- 一旦用户把数字购买理解为弱所有权，平台可能面临转向竞争生态、盗版替代、监管诉求或品牌折损。

## 9. 未解决问题

- 数字购买是否会出现可转售、可继承、可跨平台携带的标准？
- Web3 所谓“用户拥有数字资产”能否解决版权、体验、反作弊、平台服务和商业分成问题？
- 如果平台承诺长期下载但没有法律强制，用户应如何给这类数字资产定价？
